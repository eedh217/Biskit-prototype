# 기타소득 합산 화면(SPS_OI_01) - 최종 정책서

---

## 1. 화면 개요

- **화면 ID**: SPS_OI_01
- **화면명**: 기타소득 합산
- **목적**: 선택한 연도의 1월~12월 기타소득 데이터를 월별로 합산하여 조회하는 화면
- **주요 기능**: 연도 선택, 월별 합산 데이터 조회, 신고파일 다운로드, 월별 상세 화면 이동, 기타소득 엑셀 업로드
- **연관 화면**: 기타소득 월별 리스트(SPS_OI_02) — 행 클릭 시 이동
    

---

## 2. 요청한 화면 기능

### 2.1 연도 선택 셀렉박스

- 화면 상단에 연도 셀렉박스를 제공한다.
- 선택 가능 범위: **2026년 ~ (현재 연도 + 1)년**
    - 예: 현재 2026년 → 2026년, 2027년
    - 예: 현재 2028년 → 2026년, 2027년, 2028년, 2029년
- 기본 선택값: **현재 연도**
- 연도 선택 시 하단 테이블 데이터를 해당 연도 기준으로 갱신한다.
- 항상 기본값(현재 연도)이 선택되어 있으므로 placeholder("선택")는 적용하지 않는다.

### 2.2 월별 합산 테이블

- 행: 1월 ~ 12월 (기본 12행)
- 기본 정렬: 1월 → 12월 순 (월 오름차순 고정)
- 컬럼 정의:

| 컬럼명 | 표시형식 | 정렬 | 비고 |
| --- | --- | --- | --- |
| 월 | "N월" | 좌측 | 1월~12월 고정 |
| 건수(소득자 건수) | 숫자 | 우측 | 해당 월 기타소득 지급 건수 (레코드 개수) |
| 총 지급액 | 금액(천 단위 콤마 + "원") | 우측 | 해당 월 지급액 합계 |
| 총 필요경비 | 금액(천 단위 콤마 + "원") | 우측 | 해당 월 필요경비 합계 |
| 총 소득금액 | 금액(천 단위 콤마 + "원") | 우측 | 해당 월 소득금액 합계 |
| 총 소득세 | 금액(천 단위 콤마 + "원") | 우측 | 해당 월 소득세 합계 |
| 총 지방소득세 | 금액(천 단위 콤마 + "원") | 우측 | 해당 월 지방소득세 합계 |
| 총 실소득금액 | 금액(천 단위 콤마 + "원") | 우측 | 해당 월 실소득금액 합계 |
| 신고파일 최종생성일 | 날짜(YYYY-MM-DD HH:mm:ss) | 중앙 | 가장 최근 신고파일 생성 날짜. 미생성 시 "-" 표시 |
| 다운로드 | 버튼 | 중앙 | 신고파일 다운로드. 미생성 시 버튼 공간 유지 후 숨김(visibility:hidden) |

- **데이터 출처**: SPS_OI_02(해당 월 기타소득 리스트)의 개인별 지급액/필요경비/소득금액/실소득금액의 합계
- **금액 표시**: 공통 정책 5.1 적용 — 천 단위 콤마, 공통 정책 5.3 적용 — "원" 단위 표기
    - 예: 1,000,000원    
- **금액 정렬**: 공통 정책 5.2 적용 — 우측 정렬
- **음수 금액**: 음수 금액은 발생하지 않는다.
- **데이터 없는 월**: 건수 0, 금액 컬럼은 0원으로 표시

### 2.3 행 클릭 (월별 상세 화면 이동)
- 행 클릭 시 해당 월의 기타소득 리스트 화면(SPS_OI_02)으로 이동한다.
- **클릭 영역**: 다운로드 버튼 영역을 **제외한** 행 전체

### 2.4 다운로드

- 각 월별 행에 다운로드 버튼을 제공한다.
- 신고파일이 생성된 경우, 가장 최근에 생성한 신고파일을 다운로드한다.
- **신고파일 미생성 시**: 다운로드 버튼을 공간 유지 후 숨김(visibility:hidden) 처리한다. 버튼 공간은 그대로 차지한다.
- 다운로드 파일명: `YYYY년 M월 간이지급명세서.xlsx`
    - 예: `2026년 1월 간이지급명세서.xlsx`

### 2.5 엑셀 업로드

- 기타소득 합산 화면에 **엑셀 업로드** 버튼을 제공한다.
- 엑셀 파일을 통해 기타소득 데이터를 대량으로 업로드할 수 있다.
- 엑셀 업로드 결과 팝업에서 **"확인" 버튼 클릭 시** 합산 테이블 데이터를 최신 상태로 갱신한다.
- 엑셀 업로드 기능의 상세 정책은 [**기타 엑셀 업로드 정책서(policy_기타소득엑셀업로드.md)**]를 참조한다.

### 2.6 신고파일 최종생성일

- 해당 월의 모든 기타소득 건 중 reportFileDate의 최댓값을 표시한다.
    - 개별 건마다 생성일이 다를 수 있으므로 가장 최근 날짜를 표시
- **신고파일 미생성 시**: "-" 표시

### 2.7 접근 권한

- 본 화면은 **모든 로그인 사용자**가 접근할 수 있다.
- 별도의 역할/권한 제한을 적용하지 않는다.

### 2.8 로딩 상태 표시

- 연도 변경 시 또는 초기 로딩 시 데이터를 불러오는 동안 **로딩 인디케이터**(스피너)를 표시한다.

## 3. 충돌 지점 정리

| 공통 정책 항목 | 공통 정책 내용 | 본 화면 적용 | 사유 |
| --- | --- | --- | --- |
| 1.3 행 클릭 → 수정 팝업 | 행 클릭 시 수정 팝업 노출 | **변경** → 월별 상세 화면 이동 | 합산 화면이므로 수정 팝업이 아닌 해당 월 리스트 화면으로 이동 |
| 1.3 행 선택 체크박스 | 좌측 체크박스 제공(멀티 선택) | **미적용** | 선택/삭제 기능 없음 |
| 1.3 기본 정렬 | 최근 등록 순 | **변경** → 월 오름차순(1월~12월) 고정 | 월별 고정 행이므로 등록 순서 무관 |
| 2.1 셀렉박스 placeholder | 미선택 시 "선택" 표시 | **미적용** | 연도 셀렉박스는 항상 기본값(현재 연도)이 선택됨 |
