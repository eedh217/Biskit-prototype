# 직원 상세 - 연차/휴가 탭(HRIS_EMP_03_VAC) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_EMP_03_VAC
- **화면명**: 직원 상세 - 연차/휴가 탭
- **목적**: 선택한 직원의 연차 현황, 신청 내역, 이력을 조회하고 연차를 신청하는 탭
- **주요 기능**: 연차 현황 조회, 신청 내역 조회, 연차 신청, 신청 취소, 연차 이력 조회
- **진입 경로**: HRIS_EMP_03 (직원 상세) 화면에서 "연차/휴가" 탭 선택 시 표시
- **연관 화면**:
  - HRIS_EMP_03 (직원 상세) — 부모 화면
  - 연차/휴가 신청 다이얼로그 — '+ 휴가 신청' 버튼 클릭 시 노출
  - 반려 사유 다이얼로그 — '사유 보기' 버튼 클릭 시 노출
  - HRIS_LEAVE (연차/휴가 관리) — 휴가 종류 마스터 데이터 관리

|------|

### 2.3 연차 신청

#### 2.3.1 연차 신청 다이얼로그

**다이얼로그 제목**
- "연차/휴가 신청"

**다이얼로그 설명**
- "{직원명}님의 휴가를 신청합니다."

**연차 현황 요약**
- 다이얼로그 상단에 연차 현황 표시 
- 총 발생, 사용, 대기중, 잔여 (4개 항목)

#### 2.3.2 입력 필드

| 필드명 | 필수 여부 | 입력 타입 | 비고 |
|--------|-----------|-----------|------|
| 휴가 유형 | 필수 (*) | 셀렉트박스 | HRIS_LEAVE (연차/휴가 관리) 메뉴의 '휴가 종류 관리' 탭에서 관리하는 활성화된 휴가 리스트<br>(법정 휴가: 연차, 출산전후휴가, 육아휴직 등 + 회사 휴가) |
| 사용 단위 | 조건부 필수 | 셀렉트박스 | 선택한 휴가 유형의 사용 단위에 따라 표시<br>• 일 단위: "일 단위 (기간 선택)"<br>• 반일 단위: "오전 반차", "오후 반차"<br>• 시간 단위: "1시간" ~ "8시간"<br>**시간 단위 표시 조건**:<br>- 휴가 종류의 사용 단위에 '시간 단위'가 포함되어 있고<br>- 연차/휴가 시간 단위 사용 설정이 on인 경우에만 표시<br>- 설정이 off인 경우 시간 단위 옵션은 표시되지 않음 |
| 시작일 | 필수 (*) | date input | YYYYMMDD 형식으로 저장 |
| 종료일 | 조건부 필수 | date input | YYYYMMDD 형식으로 저장<br>반일/시간 단위 선택 시 시작일과 동일하게 자동 설정 후 비활성화 |
| 사유 | 필수 (*) | textarea (3행) | - |

#### 2.3.3 사용 단위 선택에 따른 동작

**일 단위 선택 시**
- 시작일과 종료일을 자유롭게 선택 가능
- 기간 범위 내 평일 기준으로 일수 계산

**반일 단위 선택 시 (오전 반차 / 오후 반차)**
- 시작일 선택 시 종료일이 자동으로 시작일과 동일하게 설정됨
- 종료일 입력 필드 비활성화 (수정 불가)
- 차감 일수: 0.5일 고정

**시간 단위 선택 시 (1시간 ~ 8시간)**
- 시작일 선택 시 종료일이 자동으로 시작일과 동일하게 설정됨
- 종료일 입력 필드 비활성화 (수정 불가)
- 차감 일수: 선택한 시간 ÷ 8
  - 예: 3시간 선택 시 0.375일 차감
  - 예: 8시간 선택 시 1일 차감

#### 2.3.4 예상 차감 일수 계산

**표시 조건**
- 휴가 유형, 사용 단위, 시작일, 종료일이 모두 입력된 경우

**표시 영역**
- 텍스트: "예상 차감 일수: N일"

**계산 로직**
- **연차를 차감하는 휴가**:
  - **일 단위**: 평일 계산 (공휴일 제외)
    - 시작일 ~ 종료일 범위의 평일(월~금) 일수 계산
    - 공휴일은 제외
    - 추가 안내문구: "(평일 기준, 공휴일 제외)"
  - **오전 반차 / 오후 반차**: 0.5일 고정
    - 추가 안내문구: "(반차는 0.5일 고정)"
  - **시간 단위**: 선택한 시간 ÷ 8
    - 예: 3시간 = 0.375일, 8시간 = 1일
    - 추가 안내문구: "(N시간 = N÷8일)"
- **연차를 차감하지 않는 휴가**:
  - 출산전후휴가, 육아휴직 등 법정 휴가
  - 예상 차감 일수 표시하지 않음 (연차에서 차감되지 않음)

#### 2.3.5 신청 버튼

**활성화 조건**
- 모든 필수 필드 입력
- 제출 중이 아닐 때

**비활성화 조건**
- 필수 필드 미입력
- 제출 중일 때

**클릭 시 동작**

1. **일수 제한 휴가 검증** (최대 사용 일수가 정해진 휴가)
   - 일수가 정해진 휴가(배우자 출산휴가 10일, 생리휴가 12일 등)인 경우
   - 해당 휴가 유형으로 **대기중 + 승인됨** 상태인 신청의 총 일수 계산
   - (이미 신청/승인된 일수 + 신청하려는 일수) > 허용 일수인 경우:
     - Alert 경고 메시지: `"{휴가명}는 최대 {허용일수}일까지 사용 가능합니다.\n이미 신청/승인: {사용일수}일\n신청 가능: {잔여일수}일"`
     - 검증 실패 시 중단
   - **적용 대상**: 법정 휴가, 회사 휴가 모두

2. **시간 기반 검증** (시간 단위 또는 반일 단위 신청 시)
   - 같은 날짜에 이미 신청/승인된 휴가가 있는 경우, 하루 최대 8시간 제한 검증
   - 시간 환산:
     - 일 단위: 8시간/일
     - 오전/오후 반차: 4시간
     - N시간: N시간
   - 검증 조건: (이미 사용한 시간 + 신청하려는 시간) > 8시간
   - 초과 시 Alert 경고 메시지: `"해당 날짜는 이미 N시간 사용 중입니다. 총 8시간까지만 신청 가능합니다."`
   - 검증 실패 시 중단
   - **적용 대상**: 반일 또는 시간 단위 신청 시 해당 날짜에 대한 검증

3. **같은 사용 단위 중복 확인** (반일 단위 신청 시)
   - 같은 날짜에 같은 사용 단위를 중복으로 신청하는지 확인
   - **적용 대상**: 오전반차, 오후반차
   - 검증 조건: 같은 날짜에 이미 **대기중 또는 승인됨** 상태인 동일 사용 단위 신청이 있는 경우
   - Alert 경고 메시지:
     - 오전반차: `"해당 날짜에 이미 오전반차를 신청했거나 승인되었습니다."`
     - 오후반차: `"해당 날짜에 이미 오후반차를 신청했거나 승인되었습니다."`
   - 검증 실패 시 중단
   - **예시**:
     - 1/15에 오전반차 승인 상태 → 같은 날 오전반차 재신청 불가
     - 1/15에 오전반차 승인 상태 → 같은 날 오후반차 신청 가능 (다른 사용 단위)

4. **날짜 중복 확인** (일 단위 신청 시)
   - 신청하려는 날짜 범위가 기존 **대기중 또는 승인됨** 상태인 휴가와 겹치는지 확인
   - 겹치는 경우:
     - Alert 경고 메시지:
       - 단일 날짜: `"mm/dd는 이미 휴가를 신청했거나 승인된 날짜입니다."`
       - 날짜 범위: `"mm/dd~mm/dd는 이미 휴가를 신청했거나 승인된 날짜입니다."`
     - 검증 실패 시 중단

5. **잔여 연차 확인** (연차를 차감하는 휴가만 해당)
   - 선택한 휴가가 연차를 차감하는 휴가인 경우에만 검증
   - 예상 차감 일수 > 잔여 연차 - 대기중 일수
   - 부족 시 Alert 경고 메시지: "잔여 연차가 부족합니다."
   - 부족 시 중단
   - **연차를 차감하지 않는 법정 휴가**는 연차 잔액 확인 건너뜀

6. **신청 처리**
   - 신청 성공 시:
     - 토스트 메시지: "휴가 신청이 완료되었습니다. (대기중)"
     - 데이터 새로고침
     - 다이얼로그 닫기
   - 신청 실패 시:
     - Alert 경고 메시지: "휴가 신청에 실패했습니다."

**제출 중일 때**
- 버튼 텍스트: "신청 중..."
- 버튼 비활성화

#### 2.3.6 취소 버튼

**클릭 시 동작**
- 입력값 초기화
- 다이얼로그 닫기

### 2.4 연차 이력

#### 2.4.1 표시 조건
- **이력 보기 토글이 ON인 경우**
- **연차/휴가 탭이 선택된 경우**
- 두 조건이 모두 만족해야 이력 섹션 표시

#### 2.4.2 이력 표시

**제목**
- "연차이력"

**타임라인 형식**
- 각 이력 항목을 시간순으로 세로로 나열
- 발생 시간 기준 내림차순 (최신순)

**이력 항목 구성**
1. **구분 및 발생시간** (상단)
   - 표시 형식: {구분} · {발생시간}
   - 구분: "발생" / "사용" / "취소" / "조정"
   - 발생시간: YYYY-MM-DD HH:mm 형식

2. **변경 내용 카드**
   - 일수: 실제 사용/발생 일수 + 휴가 이름 표시
   - 사유: 날짜 범위 또는 발생 사유

#### 2.4.3 이력 구분별 표시

**모든 휴가 종류 이력 표시**
- 연차/반차뿐만 아니라 **모든 법정 휴가, 회사 휴가의 이력도 표시**
- 연차 차감 여부에 따라 색상 및 표시 방식 구분

| 이력 구분 | 표시명 | 일수 표시 | 일수 색상 | 사유 표시 |
|-------------|--------|----------|-----------|-----------|
| 발생 | 발생 | +{일수}일 | 녹색 (+) | 이력의 사유 (예: "2024년 입사일 기준 연차 발생") |
| 사용 | 사용 | **연차 차감 있음**: -{일수}일 ({휴가명})<br>**연차 차감 없음**: {일수}일 ({휴가명}) | **연차 차감 있음**: 빨간색 (-)<br>**연차 차감 없음**: 검정색 | **연차/휴가 신청을 통해 사용한 경우: 신청 시 입력한 사유 제공**<br>그 외: 날짜 범위 |
| 취소 | 취소 | **연차 차감 있음**: +{일수}일 ({휴가명})<br>**연차 차감 없음**: {일수}일 ({휴가명}) | **연차 차감 있음**: 녹색 (+)<br>**연차 차감 없음**: 검정색 | 날짜 범위 + (승인 취소) |
| 조정 | 조정 | **차감**: -{일수}일 ({휴가명})<br>**복원**: +{일수}일 ({휴가명}) | **차감**: 빨간색 (-)<br>**복원**: 녹색 (+) | 날짜 범위 + (연차 차감 설정 변경으로 인한 조정)<br>또는<br>날짜 범위 + (연차 차감 설정 해제로 인한 복원) |

**일수 표시 상세**
- 실제 사용/발생 일수를 사용 단위와 함께 표시
- 휴가 이름도 함께 표시 (예: "연차", "배우자 출산휴가")
- **일수 표시 형식**:
  - 일 단위: `N일 (휴가명)` 예: `2일 (연차)`
  - 오전반차: `오전반차(0.5일) (휴가명)` 예: `오전반차(0.5일) (연차)`
  - 오후반차: `오후반차(0.5일) (휴가명)` 예: `오후반차(0.5일) (연차)`
  - 시간 단위: `N시간(실제일수) (휴가명)` 예: `3시간(0.375일) (연차)`
- **색상 규칙**:
  - **연차를 차감하는 휴가**:
    - 사용: `-{일수표시} (연차)` - 빨간색
      - 예: `-오전반차(0.5일) (연차)`, `-3시간(0.375일) (연차)`
    - 취소: `+{일수표시} (연차)` - 녹색
  - **연차를 차감하지 않는 휴가**:
    - 사용: `{일수표시} (휴가명)` - 검정색
      - 예: `5일 (배우자 출산휴가)`
    - 취소: `{일수표시} (휴가명)` - 검정색

**사용(use) 타입의 사유 표시 우선순위**
1. 연차/휴가 신청을 통해 사용한 경우: 신청 다이얼로그(2.3)에서 입력한 사유 제공
2. 그 외의 경우: 날짜 범위 (YYYY-MM-DD 또는 YYYY-MM-DD ~ YYYY-MM-DD)

**조정(adjust) 타입 발생 조건**
- 회사 휴가의 "연차 잔액 영향" 설정이 변경될 때 자동으로 생성됨
- **영향 없음 → 영향 있음 변경 시**:
  - 이미 승인된 해당 휴가 사용 내역에 대해 "조정(adjust)" 이력 추가 (차감)
  - 일수: `-{사용일수}` (마이너스, 빨간색)
  - 사유: `{날짜범위} (연차 차감 설정 변경으로 인한 조정)`
- **영향 있음 → 영향 없음 변경 시**:
  - 이미 승인된 해당 휴가 차감 내역에 대해 "조정(adjust)" 이력 추가 (복원)
  - 일수: `+{사용일수}` (플러스, 녹색)
  - 사유: `{날짜범위} (연차 차감 설정 해제로 인한 복원)`
- 자세한 내용은 HRIS_VAC_04 (휴가 종류 관리) 정책서 참조
