# 휴가 종류 관리 화면(HRIS_VAC_04) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_VAC_04
- **화면명**: 휴가 종류 관리
- **목적**: 법정 휴가 및 회사 휴가 종류를 관리하는 화면
- **주요 기능**: 휴가 종류 조회, 회사 휴가 추가/수정/삭제, 휴가 순서 변경, 활성화/비활성화
- **진입 경로**: 인사정보 모듈 > 연차/휴가관리 메뉴 > 휴가 종류 관리 탭 선택 시 진입
- **연관 화면**:
  - 직원 상세 - 연차/휴가 탭 (HRIS_EMP_03_VAC) — 휴가 신청 시 휴가 종류 선택

---

## 2. 요청한 화면 기능

### 2.1 레이아웃 구성

**좌측 패널 (휴가 목록)**
- 스크롤 가능
- 법정 휴가와 회사 휴가 리스트 제공

**우측 패널 (상세 정보)**
- 선택한 휴가의 상세 정보 및 편집 영역

### 2.2 좌측 패널 - 휴가 종류 목록

#### 2.2.1 상단 영역
- 총 휴가 종류 개수 제공 (총 N개)
- "+ 추가하기" 버튼 클릭 시: 2.4 휴가 추가 다이얼로그 노출

#### 2.2.2 휴가 목록

**기본 구성**
- 법정 휴가와 회사 휴가를 하나의 리스트로 통합 표시
- 섹션 구분 없이 표시 순서대로 나열
- 스크롤 가능

**기본 제공 법정 휴가 7개**
1. 연차
2. 출산전후휴가
3. 출산전후휴가(다태아)
4. 육아휴직
5. 배우자 출산휴가
6. 가족돌봄휴가
7. 생리휴가

**항목 표시**
- 휴가 이름
- 구분 뱃지:
  - 법정 휴가: "법정" 뱃지 표시
  - 회사 휴가: 뱃지 없음
  - 비활성화된 경우: "비활성" 뱃지 표시
- 일수 정보:
  - 일수가 지정된 경우: "10일" (예: 배우자 출산휴가)
  - 일수가 지정되지 않은 경우: "직원마다 부여" (예: 연차)
- 유급/무급: "유급" 또는 "무급" 표시
- 사용 단위: "일", "반일", "시간" (복수 선택 시 쉼표로 구분하여 표시)
- 활성 상태 표시: 활성화된 경우 일반 표시, 비활성화된 경우 흐림 효과
- 클릭 시: 우측 패널에 상세 정보 표시

**항목별 액션 버튼**

각 휴가 항목에는 우측에 액션 버튼이 표시됩니다:

**법정 휴가**
- **활성화/비활성화 토글 버튼**
  - 클릭 시 즉시 적용
  - 토스트 메시지:
    - 활성화: "{휴가명}이(가) 활성화되었습니다."
    - 비활성화: "{휴가명}이(가) 비활성화되었습니다."

**회사 휴가**
- **활성화/비활성화 토글 버튼**
  - 법정 휴가와 동일
- **수정 버튼**
  - 클릭 시: 2.5 휴가 수정 다이얼로그 노출
- **삭제 버튼**
  - 클릭 시:
    - Confirm 메시지: "'{휴가명}' 휴가 종류를 삭제하시겠습니까?"
    - "확인" 선택 시:
      - 휴가 삭제
      - 토스트 메시지: "휴가 종류가 삭제되었습니다."
      - 목록 새로고침
    - "취소" 선택 시: 동작 없음

**드래그앤드롭 순서 변경**
- 모든 휴가(법정 + 회사)를 하나의 리스트에서 순서 변경 가능
- 법정 휴가와 회사 휴가 간에도 순서 이동 가능
- 순서 변경 시 자동 저장

### 2.3 우측 패널 - 휴가 상세 정보 (읽기 전용)

#### 2.3.1 상세 정보 표시

**기본 구성**
- 법정 휴가와 회사 휴가 구분 없이 동일한 형식으로 표시
- 모든 항목 읽기 전용
- 수정/삭제는 좌측 리스트 아이템에서 처리

**표시 항목**

| 항목 | 표시값 | 비고 |
|------|--------|------|
| 휴가 이름 | | 텍스트 |
| 구분 | "법정 휴가" 또는 "회사 휴가" | 법정 휴가는 파란색, 회사 휴가는 회색 |
| 일수 | "10일" 또는 "직원마다 부여" | days가 null이면 "직원마다 부여" |
| 연차 잔액 영향 | "영향 있음" 또는 "영향 없음" | |
| 유급/무급 | "유급" 또는 "무급" | |
| 사용 단위 | "일 단위", "반일 단위", "시간 단위" | 복수 선택 가능, 쉼표로 구분하여 표시 |
| 활성 상태 | "활성" (녹색) 또는 "비활성" (회색) | |
| 설명 | | 없으면 "-" 표시 |

#### 2.3.2 빈 상태

**표시 조건**
- 휴가를 선택하지 않은 경우

**표시 내용**
- "휴가 종류를 선택하면 상세 정보가 표시됩니다." 메시지

### 2.4 휴가 추가 다이얼로그

#### 2.4.1 입력 필드

| 필드명 | 필수 여부 | 입력 타입 | 비고 |
|--------|-----------|-----------|------|
| 휴가 이름 | 필수 (*) | 텍스트 입력 | 최대 20자 |
| 일수 | 필수 (*) | 숫자 입력 | 1 ~ 999,999<br>회사 휴가 추가 시 필수 입력 |
| 연차 잔액 영향 | 필수 (*) | 라디오 버튼 | "영향 없음 (법정 휴가, 회사 복지 등)" (기본값) / "영향 있음 (연차 잔액에서 차감)" |
| 유급/무급 | 필수 (*) | 라디오 버튼 | "유급" (기본값) / "무급" |
| 사용 단위 | 필수 (*) | 체크박스 (복수 선택) | "일 단위" / "반일 단위" / "시간 단위"<br>최소 1개 이상 선택 필요 |
| 설명 | 선택 | 텍스트 입력 (textarea 3행) | 최대 200자 |

**참고**:
- 기본 제공 법정 휴가 중 연차는 "직원마다 부여"로 설정되어 있어 일수가 지정되지 않습니다.
- "연차 잔액 영향"을 "영향 있음"으로 선택하면 실제 신청한 일수만큼 연차 잔액에서 차감됩니다.
- 사용 단위는 복수 선택이 가능하며, 연차의 경우 일, 반일, 시간 단위를 모두 사용할 수 있습니다.

#### 2.4.2 추가 버튼

**활성화 조건**
- 모든 필수 필드가 입력되어 있을 때

**클릭 시 동작**

1. **휴가 저장**
   - 저장 성공 시:
     - 토스트 메시지: "{휴가명}이(가) 추가되었습니다."
     - 목록 새로고침
     - 다이얼로그 닫기
   - 저장 실패 시:
     - Alert 경고 메시지: {에러 메시지}

**제출 중일 때**
- 버튼 텍스트: "추가 중..."
- 버튼 비활성화

#### 2.4.3 취소 버튼

**클릭 시 동작**
- 입력값 초기화
- 다이얼로그 닫기

### 2.5 휴가 수정 다이얼로그

**적용 대상**
- 회사 휴가만 수정 가능 (법정 휴가는 수정 불가)

#### 2.5.1 입력 필드

입력 필드는 2.4.1 휴가 추가 다이얼로그와 동일하며, 기존 값이 입력되어 있습니다.

| 필드명 | 필수 여부 | 입력 타입 | 비고 |
|--------|-----------|-----------|------|
| 휴가 이름 | 필수 (*) | 텍스트 입력 | 최대 50자 |
| 일수 | 필수 (*) | 숫자 입력 | 1 ~ 999,999<br>회사 휴가 수정 시 필수 입력 |
| 연차 잔액 영향 | 필수 (*) | 라디오 버튼 | "영향 없음 (법정 휴가, 회사 복지 등)" / "영향 있음 (연차 잔액에서 차감)" |
| 유급/무급 | 필수 (*) | 라디오 버튼 | "유급" / "무급" |
| 사용 단위 | 필수 (*) | 체크박스 (복수 선택) | "일 단위" / "반일 단위" / "시간 단위"<br>최소 1개 이상 선택 필요 |
| 설명 | 선택 | 텍스트 입력 (textarea 3행) | 최대 500자 |

**참고**:
- 법정 휴가는 수정할 수 없습니다. 회사 휴가만 수정 가능합니다.
- "연차 잔액 영향"을 "영향 있음"으로 선택하면 실제 신청한 일수만큼 연차 잔액에서 차감됩니다.
- 사용 단위는 복수 선택이 가능합니다.

#### 2.5.2 저장 버튼

**활성화 조건**
- 모든 필수 필드가 입력되어 있을 때

**클릭 시 동작**

1. **"연차 잔액 영향" 변경 감지**

   **변경 여부 확인**
   - 기존 설정과 새로운 설정을 비교하여 변경 여부 확인
   - 변경되지 않은 경우: 재계산 건너뛰고 **4번 단계(휴가 수정)**로 이동
   - 변경된 경우: **2번 단계(변경 확인 다이얼로그)**로 이동

2. **변경 확인 다이얼로그 노출** ("연차 잔액 영향" 변경 시)

   **Confirm 메시지**
   - "연차 잔액 영향 변경 시, 기존에 휴가를 신청하거나 승인한 대상자의 연차 사용 현황이 변동됩니다. 변경하시겠습니까?"

   **확인 버튼 클릭 시**
   - 3번 단계(연차 재계산)로 이동

   **취소 버튼 클릭 시**
   - 다이얼로그 닫기
   - 휴가 종류 수정 다이얼로그 유지 (입력값 유지)
   - 저장 동작 중단

3. **연차 재계산 처리** ("연차 잔액 영향" 변경 시)

   **케이스 1: "영향 없음" → "영향 있음"으로 변경**

   **승인된(approved) 신청 처리:**
   - 해당 휴가 유형으로 **이미 승인됨** 상태인 모든 신청 조회
   - 각 신청에 대해:
     - 해당 직원의 연차에서 사용 일수만큼 차감
     - 연차 이력에 "조정(adjust)" 이력 추가
       - 일수: `-{사용일수}`
       - 사유: `{날짜범위} (연차 차감 설정 변경으로 인한 조정)`
   - 결과: 연차 현황의 **"사용" 일수에 반영**

   **대기중(pending) 신청 처리:**
   - 해당 휴가 유형으로 **대기중** 상태인 모든 신청 조회
   - 별도 연차 이력 추가 불필요 (아직 승인 전이므로)
   - 결과: 연차 현황의 **"대기중" 일수에 반영**

   **최종 처리:**
   - 모든 영향받는 직원의 연차 잔액 재계산
     - 사용 일수: 승인된 신청 중 연차 차감 휴가만 합산 (해당 휴가 포함)
     - 대기중 일수: 대기중 신청 중 연차 차감 휴가만 합산 (해당 휴가 포함)
     - 잔여 일수: 총 발생 - 사용 - 대기중

   **케이스 2: "영향 있음" → "영향 없음"으로 변경**

   **승인된(approved) 신청 처리:**
   - 해당 휴가 유형으로 **이미 승인됨** 상태인 모든 신청 조회
   - 각 신청에 대해:
     - 해당 직원의 연차에 사용 일수만큼 복원
     - 연차 이력에 "조정(adjust)" 이력 추가
       - 일수: `+{사용일수}`
       - 사유: `{날짜범위} (연차 차감 설정 해제로 인한 복원)`
   - 결과: 연차 현황의 **"사용" 일수에서 제외**

   **대기중(pending) 신청 처리:**
   - 해당 휴가 유형으로 **대기중** 상태인 모든 신청 조회
   - 별도 연차 이력 추가 불필요 (아직 승인 전이므로)
   - 결과: 연차 현황의 **"대기중" 일수에서 제외**

   **최종 처리:**
   - 모든 영향받는 직원의 연차 잔액 재계산
     - 사용 일수: 승인된 신청 중 연차 차감 휴가만 합산 (해당 휴가 제외)
     - 대기중 일수: 대기중 신청 중 연차 차감 휴가만 합산 (해당 휴가 제외)
     - 잔여 일수: 총 발생 - 사용 - 대기중

4. **휴가 수정**
   - 수정 성공 시:
     - 토스트 메시지: "휴가 종류가 수정되었습니다."
     - 목록 새로고침
     - 다이얼로그 닫기
   - 수정 실패 시:
     - Alert 경고 메시지: {에러 메시지}

#### 2.5.3 취소 버튼

**클릭 시 동작**
- 변경 사항 취소
- 다이얼로그 닫기

## 3. 데이터 정책

### 3.1 법정 휴가 기본 데이터

| 휴가명 | 일수 | 유급 | 사용단위 | 차감일수 | 설명 |
|--------|------|------|---------|---------|------|
| 연차 | 직원마다 부여 | 유급 | 일, 반일, 시간 단위 | - | 직원마다 부여되는 연차입니다. 일 단위, 반일 단위, 시간 단위로 사용 가능합니다. |
| 출산전후휴가 | 90일 | 유급 | 1일 단위 | 0일 | 출산 전후 90일의 휴가입니다. |
| 출산전후휴가(다태아) | 120일 | 유급 | 1일 단위 | 0일 | 다태아 출산 시 120일의 휴가입니다. |
| 육아휴직 | 365일 | 무급 | 1일 단위 | 0일 | 만 8세 이하 또는 초등학교 2학년 이하 자녀 양육을 위한 휴직입니다. |
| 배우자 출산휴가 | 10일 | 유급 | 1일 단위 | 0일 | 배우자 출산 시 10일의 유급휴가입니다. |
| 가족돌봄휴가 | 10일 | 무급 | 1일 단위 | 0일 | 가족의 질병, 사고, 노령 등으로 돌봄이 필요한 경우 연간 최대 10일까지 사용 가능합니다. |
| 생리휴가 | 12일 | 무급 | 1일 단위 | 0일 | 여성 근로자에게 월 1일씩 연간 12일이 부여됩니다. |

### 3.2 연차 차감 일수 정책

**연차를 차감하는 휴가**
- 연차 현황의 "사용" 및 "대기중" 일수에 포함
- 연차 이력에 차감/복원 기록 생성
- 예: 연차 (사용 단위에 따라 차감 - 일 단위: 1일, 반일 단위: 0.5일, 시간 단위: 사용 시간/8)

**연차를 차감하지 않는 휴가**
- 연차 현황 계산에 포함되지 않음
- 연차 이력에는 기록되지만 차감 없이 사용 이력만 표시
- 예: 출산전후휴가, 육아휴직, 배우자 출산휴가 등

**"연차 잔액 영향" 설정 변경 시**
- 회사 휴가의 "연차 잔액 영향" 설정을 변경하면 해당 휴가로 이미 승인된 모든 신청에 대해 연차 재계산 수행
- **영향 없음 → 있음**: 기존 승인된 사용 내역에 대해 연차 차감 처리, 연차 이력에 "사용(use)" 이력 추가
- **영향 있음 → 없음**: 기존 차감된 연차 복원 처리, 연차 이력에 "조정(adjust)" 이력 추가
- 모든 영향받는 직원의 연차 잔액이 자동으로 재계산됨

### 3.3 활성화/비활성화 정책

**비활성화된 휴가**
- 휴가 신청 시 선택 불가 (셀렉트박스에 표시되지 않음)
- 기존에 신청/승인된 내역은 유지
- 관리 화면에서는 표시되지만 흐림 효과로 구분

**활성화된 휴가**
- 휴가 신청 시 선택 가능
- 정상적으로 표시 및 사용 가능
