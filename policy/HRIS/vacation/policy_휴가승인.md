# 휴가 승인 화면(HRIS_VAC_02) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_VAC_02
- **화면명**: 휴가 승인
- **목적**: 직원들이 신청한 휴가를 승인하거나 반려하는 화면
- **주요 기능**: 대기중인 휴가 신청 목록 조회, 승인, 반려 (사유 입력)
- **진입 경로**: 인사정보 모듈 > 연차/휴가관리 메뉴 > 휴가 승인 탭 선택 시 진입
- **연관 화면**:
  - HRIS_VAC_02_DLG (반려 사유 입력 Dialog) — 반려 버튼 클릭 시 팝업

---

## 2. 요청한 화면 기능

### 2.1 헤더 영역

- **제목**: "대기중인 휴가 신청"
- **대기 건수 Badge**:
  - 표시 형식: "N건" (예: "5건")

### 2.2 리스트(테이블)

- **조회 대상**: 승인 대기중인 휴가 신청만 (상태='대기중')
- **정렬**: 신청일시 기준 (먼저 신청한 순)
- **체크박스**: 제공하지 않음
- **행 클릭**: 제공하지 않음
- 컬럼 정의:

| 컬럼명 | 표시형식 | 비고 |
|--------|---------|------|
| 신청일 | YYYY. M. D. | |
| 신청자 | 텍스트 | 직원명 표시|
| 유형 | 텍스트 | 연차, 반차(오전), 반차(오후) |
| 기간 | YYYY-MM-DD ~ YYYY-MM-DD | 시작일 ~ 종료일 |
| 일수 | "N일" | 평일 기준 실제 차감 일수 |
| 사유 | 텍스트 | hover 시 전체 내용 표시 (title 속성) |
| 액션 | 버튼 2개 | 승인 버튼 + 반려 버튼 |

### 2.3 액션 버튼

#### 승인 버튼
- **버튼명**: "승인"
- **클릭 시**:
  1. confirm 노출
     - 문구: "휴가 신청을 승인하시겠습니까?"
  2. 확인 선택 시:
     - 해당 신청의 상태를 '승인'으로 변경
     - 신청자의 연차 잔액에서 일수 자동 차감
     - 토스트 노출: "휴가 신청이 승인되었습니다."
     - 리스트 즉시 새로고침 (해당 신청은 목록에서 제거됨)
  3. 취소 선택 시: 아무 동작 없음

#### 반려 버튼
- **버튼명**: "반려"
- **클릭 시**: 반려 사유 입력 Dialog 열림 (HRIS_VAC_02_DLG)

### 2.4 반려 사유 입력 Dialog (HRIS_VAC_02_DLG)

#### Dialog 헤더
- **제목**: "휴가 신청 반려"

#### Dialog 내용
- **반려 사유 입력**:
  - 입력 컴포넌트: Textarea (4줄)
  - placeholder: "반려 사유를 입력하세요"
  - **필수 입력**: 빈 값 또는 공백만 입력 시 제출 불가

#### Dialog 액션
1. **취소 버튼**
   - 버튼명: "취소"
   - 클릭 시: Dialog 닫기 (입력 내용 초기화)

2. **반려 버튼**
   - 버튼명: "반려"
   - **활성화 조건**: 반려 사유가 입력된 경우에만 활성화 (빈 값 또는 공백만 있으면 비활성화)
   - 클릭 시:
     - **검증**: 반려 사유가 빈 값이거나 공백만 있는 경우
       - 토스트 노출: "입력 오류 - 반려 사유를 입력해주세요."
       - Dialog 유지
     - **정상 처리**: 반려 사유가 입력된 경우
       - 해당 신청의 상태를 '반려'로 변경
       - 반려 사유 저장
       - 연차 잔액 변동 없음
       - 토스트 노출: "휴가 신청이 반려되었습니다."
       - Dialog 닫기
       - 리스트 즉시 새로고침 (해당 신청은 목록에서 제거됨)

### 2.5 빈 화면 처리

#### 로딩 중
- 표시 문구: "로딩 중..."
- 스타일: 기본 텍스트

#### 데이터 0건
- 표시 문구: "대기중인 휴가 신청이 없습니다."
- 스타일:
  - 흰색 배경, 테두리, 라운드 처리
  - 중앙 정렬 (text-center)
  - 패딩: p-12
  - 텍스트 색상: text-gray-500

## 3. 충돌 지점 정리

| 공통 정책 항목 | 공통 정책 내용 | 본 화면 적용 | 사유 |
|--------------|--------------|------------|------|
| 1.1 검색 | 검색 기능 | **미적용** | 대기중인 신청만 표시, 검색 불필요 |
| 1.2 리스트 개수 | 카운트 표시 | **부분 적용** | Badge로 건수 표시 (N건) |
| 1.3 리스트(테이블) | 체크박스, 행 클릭 | **미적용** | 체크박스 없음, 행 클릭 없음 |
| 1.4 페이지 사이즈 | 페이지네이션 | **미적용** | 전체 신청 한 화면에 표시 (페이징 없음) |
| 1.5 엑셀 다운로드 | 다운로드 기능 | **미적용** | 엑셀 다운로드 기능 없음 |
| 1.6 선택 삭제 | 삭제 기능 | **미적용** | 삭제 기능 없음 |

> 본 화면은 승인 업무에 특화된 화면으로, 대기중인 신청만 조회하며 승인/반려 액션에 집중한다. 검색, 페이징, 엑셀 다운로드 등의 부가 기능은 제공하지 않는다.
