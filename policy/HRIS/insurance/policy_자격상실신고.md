# 자격상실신고 화면(HRIS_INS_02) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_INS_02
- **화면명**: 자격상실신고
- **목적**: 4대보험(국민연금, 건강보험, 고용보험, 산재보험) 자격상실 신고 및 관리
- **주요 기능**: 사업장 정보 입력, 직장가입자 추가/삭제, 보험별 상세 정보 입력, 임시 저장, 신고내역 조회, 신고하기
- **진입 경로**: 인사정보 모듈 > 4대보험 > 자격상실신고 메뉴 선택 시 진입
- **연관 화면**: 없음 (단일 화면에서 모든 작업 완료)

---

## 2. 요청한 화면 기능

### 2.1 사업장 정보

#### 2.1.1 입력 항목

**필수 입력 항목**:
- 사업장관리번호 *
- 명칭 *
- 우편번호 *
- 주소 *
- 상세주소 *

**선택 입력 항목**:
- 전화번호
- FAX번호

#### 2.1.2 입력 검증 규칙

**전화번호**:
- 허용 문자: 숫자(0-9), +, (, ), -
- 정규식: `/^[0-9+()-]*$/`
- 입력 예시: `02-1234-5678`, `+82-2-1234-5678`, `02)1234-5678`
- 허용되지 않은 문자 입력 시: 입력되지 않음 (실시간 검증)

**FAX번호**:
- 허용 문자: 숫자(0-9), -
- 정규식: `/^[0-9-]*$/`
- 입력 예시: `02-1234-5678`
- 허용되지 않은 문자 입력 시: 입력되지 않음 (실시간 검증)

#### 2.1.3 주소 검색 기능

- 우편번호/주소 필드는 직접 입력 불가 (readOnly)
- "주소 검색" 버튼 클릭 시 주소 검색 다이얼로그 오픈
- Daum 우편번호 서비스 API 사용
- 주소 선택 완료 시 우편번호와 주소 자동 입력
- 상세주소만 사용자가 직접 입력

#### 2.1.4 사업장 정보 저장 및 동기화

- 신고 완료 시 사업장 정보를 LocalStorage에 저장
- 다음 신고 시 자동으로 불러오기 (편의성 제공)
- **전체 연동**: 이 사업장 정보는 4대보험관리 메뉴 전체에서 공유됩니다
  - 자격 취득신고
  - 자격 상실신고
  - 보수월액 변경
  - 기타 4대보험 관련 메뉴
- 한 곳에서 사업장 정보를 입력하면 다른 메뉴에서도 동일한 정보가 자동으로 표시됩니다

### 2.2 직장가입자 목록

#### 2.2.1 직원 추가/삭제

**초기 상태**:
- 화면 진입 시 기본 직원 1명이 자동 추가됨

**추가 기능**:
- 버튼명: "추가하기"
- 위치: 카드 헤더 우측
- 클릭 시 직장가입자 목록 최하단에 빈 직원 카드 1개 추가

**삭제 기능**:
- 각 직원 카드 우측 상단에 휴지통 아이콘 버튼 제공
- 클릭 시 해당 직원 카드 즉시 삭제 (confirm 없음)
- 최소 1명은 유지 (전체 삭제 불가)
- 첫 번째 직원(직원1)의 삭제 버튼은 표시하지 않음

#### 2.2.2 직원 정보 입력

**기본 정보**:
- 성명 * (EmployeeCombobox 사용)
  - 직원 관리에 등록된 직원 목록에서 선택
  - 선택 시 주민등록번호, 전화번호(휴대전화), 상실연월일 자동 입력
  - **중복 방지**: 이미 추가된 직원은 선택 목록에서 제외
  - **데이터 덮어쓰기**: 직원 선택 시 기존에 수동으로 입력한 데이터가 있어도 확인 없이 덮어씌웁니다
- 주민등록번호/외국인등록번호 * (자동 입력, 수정 가능)
  - 형식: 000000-0000000 (숫자만 입력, 하이픈 자동 추가)
  - 13자리 제한
- 전화번호(휴대전화) * (자동 입력, 수정 가능)
  - 허용 문자: 숫자(0-9), +, (, ), -
  - 정규식: `/^[0-9+()-]*$/`
- 상실연월일 * (자동 입력, 수정 가능)
  - 직원의 퇴사일 자동 입력
  - 날짜 선택 UI (date picker)

### 2.3 보험 신청

#### 2.3.1 보험 종류 선택

- 4개 보험 모두 기본 체크 상태
- 체크박스로 선택/해제 가능
- 국민연금
- 건강보험
- 고용보험
- 산재보험
- **데이터 초기화**: 보험 체크 해제 시 해당 보험의 입력 데이터는 모두 초기화됩니다

#### 2.3.2 국민연금 (선택 시 표시)

**필수 입력 항목**:
- 상실부호 *
  - 1 - 사망
  - 3 - 사용관계 종료
  - 4 - 국적 상실(국외 이주)
  - 5 - 60세 도달
  - 6 - 다른 공적연금 가입
  - 9 - 전출(통ㆍ폐합)
  - 15 - (조기)노령연금 수급권 취득
  - 16 - 협정국 연금가입
  - 19 - 체류기간 만료(외국인)
  - 20 - 적용제외 체류자격(외국인)
  - 21 - 무보수 대표이사
  - 22 - 근로자 제외
  - 26 - 취득취소(착오나 사정변경으로 자격취득 신고를 취소하는 경우)

**선택 입력 항목**:
- 상실 초일부터 납부의사 (체크박스)

#### 2.3.3 건강보험 (선택 시 표시)

**필수 입력 항목**:
- 상실부호 *
  - 1 - 퇴직
  - 2 - 사망
  - 4 - 의료급여수급권자
  - 10 - 유공자등 건강보험 적용배제신청
  - 13 - 그 밖의 사유(외국인 당연적용 제외 등)
  - 16 - 취득취소(착오나 사정변경으로 자격취득 신고를 취소하는 경우)
  - 17 - 국적상실
  - 19 - 이민출국
  - 24 - 가입제외(외국의 법령)
  - 25 - 가입제외(외국의 보험)
  - 26 - 가입제외(사용자와의 계약)
  - 58 - 무보수대표자
- 연간보수총액(해당연도) *
  - 숫자만 입력 가능
  - 천 단위 콤마 자동 추가
  - 최대 12자리
- 근무개월수(해당연도) *
  - 숫자만 입력 가능
  - 최소 1개월

**선택 입력 항목**:
- 전년도 연말정산을 실시하지 않은 경우 (체크박스)
  - 체크 시 추가 필드 표시 (필수입력):
    - 연간보수총액(전년도) *
      - 숫자만 입력 가능
      - 천 단위 콤마 자동 추가
      - 최대 12자리
    - 근무개월수(전년도) *
      - 숫자만 입력 가능
      - 최소 1개월

#### 2.3.4 고용보험·산재보험 (선택 시 표시)

**표시 조건**:
- 고용보험 또는 산재보험 중 하나라도 체크된 경우 표시

**필수 입력 항목**:
- 상실사유코드 *
  - 11 - 개인사정으로 인한 자진퇴사
  - 12 - 사업장 이전, 근로조건 변동, 임금체불 등으로 자진퇴사
  - 22 - 폐업ㆍ도산
  - 23 - 경영상 필요 및 회사불황으로 인한 인원감축 등에 따른 퇴사(해고ㆍ권고사직ㆍ명예퇴직 포함)
  - 26 - 근로자 귀책사유에 의한 징계해고ㆍ권고사직ㆍ계약해지
  - 31 - 정년
  - 32 - 계약기간 만료, 공사 종료
  - 41 - 고용보험 비적용
  - 42 - 이중고용
- 구체적 사유 *
  - 텍스트 입력 (최대 50자)
  - placeholder: "예: 사업장 이전으로 인한 자진퇴사"

**선택 입력 항목 (체크박스)**:

1. **"전보 또는 휴직 등의 사유로 고용보험과 산재보험 보수총액에 차이가 있나요?"**
   - **표시 조건**: 고용보험과 산재보험을 **둘 다** 체크한 경우에만 표시
   - 체크 해제 시: 보수총액 필드 1개로 통합 (고용/산재 동일 금액)
   - 체크 시: 보수총액 필드가 보험별로 분리 (고용/산재 각각 입력)

2. **"전년도 보수총액을 신고하지 않았나요?"**
   - 항상 표시
   - 체크 해제 시: 해당연도 보수총액만 입력
   - 체크 시: 전년도 보수총액 필드 추가 표시

**보수총액 입력 필드 구성**:

아래 4가지 경우에 따라 표시되는 필드가 달라집니다:

**Case 1: 보수 차이 없음 + 전년도 신고 안 함 체크 안 함**
- 보수총액(해당연도) *
  - 숫자만 입력 가능, 천 단위 콤마 자동 추가, 최대 12자리
  - 입력 시 고용보험과 산재보험에 동일한 값 적용

**Case 2: 보수 차이 없음 + 전년도 신고 안 함 체크함**
- 보수총액(해당연도) *
- 보수총액(전년도) *
  - 각각 숫자만 입력 가능, 천 단위 콤마 자동 추가, 최대 12자리
  - 입력 시 고용보험과 산재보험에 동일한 값 적용

**Case 3: 보수 차이 있음 + 전년도 신고 안 함 체크 안 함**
- 고용보험 보수총액(해당연도) *
- 산재보험 보수총액(해당연도) *
  - 각각 숫자만 입력 가능, 천 단위 콤마 자동 추가, 최대 12자리

**Case 4: 보수 차이 있음 + 전년도 신고 안 함 체크함**
- 고용보험 보수총액(해당연도) *
- 고용보험 보수총액(전년도) *
- 산재보험 보수총액(해당연도) *
- 산재보험 보수총액(전년도) *
  - 각각 숫자만 입력 가능, 천 단위 콤마 자동 추가, 최대 12자리

### 2.4 필수값 검증 및 버튼 활성화

#### 2.4.1 필수값 표시

- 모든 필수 입력 항목 라벨 옆에 `*` 표시
- 조건부 필수 항목도 조건 충족 시 `*` 표시

#### 2.4.2 신고하기 버튼 활성화

- 필수값이 모두 입력되지 않으면: 버튼 **비활성화**
- 필수값이 모두 입력되면: 버튼 **활성화**

### 2.5 임시 저장

#### 2.5.1 저장 기능

- 버튼명: "임시 저장"
- 위치: 하단 고정 영역 우측
- 클릭 시 현재 입력된 모든 데이터를 LocalStorage에 저장
- 저장 성공 시 토스트 1.5초 노출 — 문구: "임시 저장되었습니다."
- 저장 시각 함께 저장 (ISO 형식)

#### 2.5.2 불러오기 기능

- 버튼명: "임시저장 불러오기"
- 위치: 하단 고정 영역 좌측
- 임시 저장 데이터가 있을 때만 표시
- **확인 절차**: 현재 작성 중인 데이터가 있을 경우 확인 메시지 표시
  - 메시지: "임시저장 데이터를 불러올 경우, 현재 입력한 데이터는 사라집니다. 불러오시겠습니까?"
  - 취소 클릭 시: 아무 동작 없음
  - 확인 클릭 시: 데이터 복원 진행
- 클릭 시 저장된 데이터를 폼에 복원
- 복원 성공 시 토스트 1.5초 노출 — 문구: "임시 저장 데이터를 불러왔습니다."

#### 2.5.3 삭제 기능

- 버튼명: "임시저장 삭제"
- 위치: 하단 고정 영역 좌측
- 임시 저장 데이터가 있을 때만 표시
- 클릭 시 임시 저장 데이터 삭제
- 삭제 성공 시 토스트 1.5초 노출 — 문구: "임시 저장 데이터를 삭제했습니다."
- 삭제 후 UI 변경:
  - "임시저장 불러오기" 버튼 미노출
  - "임시저장 삭제" 버튼 미노출
  - 저장 시각 표시 미노출

#### 2.5.4 저장 시각 표시

- 표시 위치: 하단 고정 영역 좌측
- 표시 형식: "마지막 저장: YYYY-MM-DD HH:mm:ss"
- 임시 저장 데이터가 있을 때만 표시

### 2.7 신고하기

#### 2.7.1 신고 확인 다이얼로그

**오픈 조건**:
- 필수값이 모두 입력된 경우에만 버튼 활성화
- 버튼 클릭 시 확인 다이얼로그 오픈

**다이얼로그 내용**:
- 제목: "자격상실 신고 확인"
- 신고 대상: N명 (이름(사번) 형식으로 리스트 표시)
  - 스크롤 가능
- 사업장 정보:
  - "사업장: [주소 표시 규칙]"
    - 서울특별시/광역시: 구까지 표시 (예: 서울특별시 강남구, 부산광역시 해운대구)
    - 도 지역: 시까지 표시 (예: 경기도 성남시)
  - 사업장 정보에 맞춰서 공단 FAX 번호 제공
- 버튼:
  - 취소: 다이얼로그 닫기
  - 확인: 신고 처리

#### 2.7.2 신고 처리

**신고 완료 시**:
1. 신고내역 저장되고 신고내역 조회 다이얼로그에 내역 추가
   - 저장 데이터: 신고일시, 사업장 정보, 직원 목록
2. 사업장 정보 저장 (다음 신고 시 사용)
3. 임시 저장 데이터 삭제
4. 토스트 1.5초 노출 — 문구: "자격상실 신고가 완료되었습니다."
5. 폼 초기화 (기본 직원 1명 추가된 상태로)

### 2.8 신고내역 조회

#### 2.8.1 다이얼로그 오픈

- 버튼명: "신고내역 조회"
- 클릭 시 신고내역 조회 다이얼로그 오픈

#### 2.8.2 기간 검색

**기본 설정**:
- 다이얼로그 오픈 시 기본 기간: 최근 1개월

**날짜 선택**:
- 시작일 / 종료일 직접 선택 (date input)
- 날짜 변경 시 기간 버튼 선택 해제

**기간 버튼**:
- 당일 / 일주일 / 1개월 / 3개월 / 6개월 / 1년
- 클릭 시 해당 기간으로 날짜 자동 설정 (종료일은 당일로 지정)

#### 2.8.3 검색어 검색

- 검색 항목: 사번 또는 성명
- placeholder: "사번 또는 성명"
- Enter 입력하거나 검색 버튼 클릭 시 검색 실행
- 검색 시 첫 페이지로 이동

#### 2.8.4 신고내역 리스트

**컬럼 구성**:
- 신고일자 (YYYY-MM-DD HH:mm:ss)
- 사업장명
- 직원 (N명(이름1, 이름2, ...))
- 미리보기 버튼

**리스트 개수**:
- 표시 위치: 리스트 상단
- 표시 형식: "총 N건"

**페이지네이션**:
- 페이지당 10개 항목
- 스마트 페이지네이션 적용
  - 7페이지 이하: 모든 페이지 번호 표시
  - 7페이지 초과: 1 ... 5 6 7 ... 20 형식
  - 현재 페이지 앞뒤 1페이지씩 표시
  - 첫 페이지와 마지막 페이지 항상 표시

**미리보기 버튼**:
- 각 행에 "미리보기" 버튼 제공
- 클릭 시 새탭 열려서 해당 신고 내역 상세 제공

**데이터 없음**:
- 신고내역이 없을 경우: "신고내역이 없습니다." 문구 중앙 표시
