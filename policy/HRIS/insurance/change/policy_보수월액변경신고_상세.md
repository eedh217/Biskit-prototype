# 보수월액 변경신고 상세 화면(HRIS_INS_CHG_03) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_INS_CHG_03
- **화면명**: 보수월액 변경신고 상세
- **목적**: 신고 완료된 보수월액 변경신고 내역을 조회하는 읽기 전용 화면
- **주요 기능**: 신고 정보 조회, 사업장 정보 조회, 직원별 신고 상세 내역 조회
- **진입 경로**:
  - HRIS_INS_CHG_01 신고완료 상태 행 클릭 → 현재 탭에서 이동
  - HRIS_INS_CHG_02 신고 완료 처리 후 → 자동 이동
- **연관 화면**:
  - HRIS_INS_CHG_01 (보수월액 변경신고 목록) — 뒤로가기 클릭 시 이동

---

## 2. 레이아웃 구조

- **상단**: PageHeader (뒤로가기 + 화면명)
- **중앙**: 좌측 패널 + 우측 패널 (좌우 분할 레이아웃)
  - **좌측 패널**: 신고 정보 + 사업장 정보 + 직원 목록
  - **우측 패널**: 선택된 직원의 신고 상세 내역 (읽기 전용)
- 수정 기능 없음 — 전체 화면 읽기 전용

---

## 3. 좌측 패널

좌측 패널은 스크롤 가능하며, 신고 정보 → 사업장 정보 → 직원 목록 순서로 표시.

### 3.1 신고 정보

| 항목 | 표시 내용 |
|------|----------|
| 신고일자 | yyyy-MM-dd HH:mm:ss |
| 상태 | 작성중 / 신고완료 (일반 텍스트) |
| 팩스발송 | 작성중 상태: "-" / 신고완료 상태: "발송성공" (일반 텍스트) |
| 사업장 FAX | 신고 시 입력한 사업장 FAX 번호 (신고완료 상태에서만 표시) |
| 공단 FAX | 신고 시 입력한 공단 FAX 번호 (신고완료 상태에서만 표시) |

### 3.2 사업장 정보

| 항목 | 표시 조건 |
|------|----------|
| 관리번호 | 항상 표시 |
| 명칭 | 항상 표시 |
| 주소 | 항상 표시 (주소 + 상세주소 합쳐서 표시) |
| 전화번호 | 항상 표시 |
| 전자우편주소 | 값이 있는 경우에만 표시 |
| 휴대폰번호 | 값이 있는 경우에만 표시 |

### 3.3 직원 목록

- 신고에 포함된 직원 목록을 순서대로 표시
- 각 항목: 성명 + 사번 표시
- 클릭 시 해당 직원이 선택 상태가 되어 우측 패널에 상세 내역 표시
- **초기 선택 상태**: 첫 번째 직원이 기본 선택됨

---

## 4. 우측 패널 — 직원 신고 상세

선택된 직원의 신고 내용을 섹션별로 표시. 모든 항목 읽기 전용.

### 4.1 기본 정보

| 항목 | 표시 조건 |
|------|----------|
| 성명 | 항상 표시 |
| 주민등록번호/외국인등록번호 | 항상 표시 |
| 보수 변경 월 | 항상 표시 |
| 변경사유 | 항상 표시 |

### 4.2 국민연금 (국민연금 신청한 경우에만 표시)

| 항목 | 표시 조건 |
|------|----------|
| 현재 기준소득월액 | 값이 0이 아닌 경우에만 표시 (천 단위 콤마 + "원" 단위 표시) |
| 변경 기준소득월액 | 값이 0이 아닌 경우에만 표시 (천 단위 콤마 + "원" 단위 표시) |

### 4.3 건강보험 (건강보험 신청한 경우에만 표시)

| 항목 | 표시 조건 |
|------|----------|
| 변경 후 보수월액 | 값이 0이 아닌 경우에만 표시 (천 단위 콤마 + "원" 단위 표시) |

### 4.4 고용보험·산재보험 (고용보험 또는 산재보험 신청한 경우에만 표시)

| 항목 | 표시 조건 |
|------|----------|
| 변경 후 월평균보수(고용보험) | 고용보험 신청 시, 값이 0이 아닌 경우에만 표시 (천 단위 콤마 + "원" 단위 표시) |
| 변경 후 월평균보수(산재보험) | 산재보험 신청 시, 고용·산재 보수가 다른 경우에만 별도 표시 (천 단위 콤마 + "원" 단위 표시) |

## 5. 충돌 지점 정리

| 항목 | 내용 | 비고 |
|------|------|------|
| 조건부 항목 표시 | 값이 없거나 0인 항목은 항목 자체를 미노출 | 빈 값 "-" 표시 방식 미사용 |
| 사업장 정보 항목 | 전자우편주소·휴대폰번호 조건부 표시 | 자격취득신고 상세와 다름 (단위사업소·영업소 없음) |
