# 자격취득신고 상세 화면(HRIS_INS_ACQ_03) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_INS_ACQ_03
- **화면명**: 자격취득신고 상세
- **목적**: 신고 완료된 자격취득신고 내역을 조회하는 읽기 전용 화면
- **주요 기능**: 신고 정보 조회, 사업장 정보 조회, 직원별 신고 상세 내역 조회
- **진입 경로**:
  - HRIS_INS_ACQ_01 신고완료 상태 행 클릭 → 현재 탭에서 이동
  - HRIS_INS_ACQ_02 신고 완료 처리 후 → 자동 이동
- **연관 화면**:
  - HRIS_INS_ACQ_01 (자격취득신고 목록) — 뒤로가기 클릭 시 이동

---

## 2. 레이아웃 구조

- **상단**: PageHeader (뒤로가기 + 화면명)
- **중앙**: 좌측 패널 + 우측 패널 (좌우 분할 레이아웃)
  - **좌측 패널**: 신고 정보 + 사업장 정보 + 직원 목록
  - **우측 패널**: 선택된 직원의 신고 상세 내역 (읽기 전용)
- 수정 기능 없음 — 전체 화면 읽기 전용

---

## 3. 좌측 패널

좌측 패널은 스크롤 가능하며, 신고 정보 → 사업장 정보 → 직원 목록 순서로 표시.

### 3.1 신고 정보

| 항목 | 표시 내용 |
|------|----------|
| 신고일자 | yyyy-MM-dd HH:mm:ss |
| 상태 | 작성중 / 신고완료 (일반 텍스트) |
| 팩스발송 | 작성중 상태: "-" / 신고완료 상태: "발송성공" (일반 텍스트) |
| 사업장 FAX | 신고 시 입력한 사업장 FAX 번호 |
| 공단 FAX | 신고 시 입력한 공단 FAX 번호 |

### 3.2 사업장 정보

| 항목 | 표시 조건 |
|------|----------|
| 관리번호 | 항상 표시 |
| 명칭 | 항상 표시 |
| 단위사업소 | 값이 있는 경우에만 표시 |
| 영업소 | 값이 있는 경우에만 표시 |
| 주소 | 항상 표시 (주소 + 상세주소 합쳐서 표시) |
| 전화번호 | 항상 표시 |

### 3.3 직원 목록

- 신고에 포함된 직원 목록을 순서대로 표시
- 각 항목: 성명 + 사번 표시
- 클릭 시 해당 직원이 선택 상태가 되어 우측 패널에 상세 내역 표시
- **초기 선택 상태**: 첫 번째 직원이 기본 선택됨

## 4. 우측 패널 — 직원 신고 상세

선택된 직원의 신고 내용을 섹션별로 표시. 모든 항목 읽기 전용.

### 4.1 기본 정보

| 항목 | 표시 조건 |
|------|----------|
| 성명 | 항상 표시 |
| 주민등록번호/외국인등록번호 | 항상 표시 |
| 국적 | 항상 표시 (국가명으로 표시) |
| 체류자격 | 외국인인 경우에만 표시 (값이 있는 경우) |
| 월 소득액 | 항상 표시 (천 단위 콤마 + "원" 단위 표시) |
| 자격취득일 | 항상 표시 |
| 대표자 여부 | 대표자인 경우에만 표시 |

### 4.2 국민연금 (국민연금 신청한 경우에만 표시)

| 항목 | 표시 조건 |
|------|----------|
| 자격취득부호 | 항상 표시 (코드명으로 표시) |
| 특수직종부호 | 값이 있는 경우에만 표시 (코드명으로 표시) |
| 직역연금부호 | 값이 있는 경우에만 표시 (코드명으로 표시) |
| 취득 월 납부 희망 | 희망한 경우에만 표시 |

### 4.3 건강보험 (건강보험 신청한 경우에만 표시)

| 항목 | 표시 조건 |

| 보험료감면부호 | 항상 표시 (코드명으로 표시) |
| 공무원·교직원 | 해당하는 경우에만 표시 |
| 회계명 | 공무원·교직원 해당 시, 값이 있는 경우 표시 |
| 회계부호 | 공무원·교직원 해당 시, 값이 있는 경우 표시 |
| 직종명 | 공무원·교직원 해당 시, 값이 있는 경우 표시 |
| 직종부호 | 공무원·교직원 해당 시, 값이 있는 경우 표시 |

| 외국인 | 외국인인 경우에만 표시 |

| 항목 | 내용 | 비고 |
|------|------|------|
| 코드값 표시 방식 | 코드 번호가 아닌 코드명으로 표시 | 예: "01" → "18세 이상 당연취득" |
