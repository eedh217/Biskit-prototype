# 자격상실신고 상세 화면(HRIS_INS_LOS_03) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_INS_LOS_03
- **화면명**: 자격상실신고 상세
- **목적**: 신고 완료된 자격상실신고 내역을 조회하는 읽기 전용 화면
- **주요 기능**: 신고 정보 조회, 사업장 정보 조회, 직원별 신고 상세 내역 조회
- **진입 경로**:
  - HRIS_INS_LOS_01 신고완료 상태 행 클릭 → 현재 탭에서 이동
  - HRIS_INS_LOS_02 신고 완료 처리 후 → 자동 이동
- **연관 화면**:
  - HRIS_INS_LOS_01 (자격상실신고 목록) — 뒤로가기 클릭 시 이동

---

## 2. 레이아웃 구조

- **상단**: PageHeader (뒤로가기 + 화면명)
- **중앙**: 좌측 패널 + 우측 패널 (좌우 분할 레이아웃)
  - **좌측 패널**: 신고 정보 + 사업장 정보 + 직원 목록
  - **우측 패널**: 선택된 직원의 신고 상세 내역 (읽기 전용)
- 수정 기능 없음 — 전체 화면 읽기 전용

---

## 3. 좌측 패널

좌측 패널은 스크롤 가능하며, 신고 정보 → 사업장 정보 → 직원 목록 순서로 표시.

### 3.1 신고 정보

| 항목 | 표시 내용 |
|------|----------|
| 신고일자 | yyyy-MM-dd HH:mm:ss |
| 상태 | 작성중 / 신고완료 (일반 텍스트) |
| 팩스발송 | 작성중 상태: "-" / 신고완료 상태: "발송성공" (일반 텍스트) |
| 사업장 FAX | 신고 시 입력한 사업장 FAX 번호 |
| 공단 FAX | 신고 시 입력한 공단 FAX 번호 |

### 3.2 사업장 정보

| 항목 | 표시 조건 |
|------|----------|
| 관리번호 | 항상 표시 |
| 명칭 | 항상 표시 |
| 주소 | 항상 표시 (주소 + 상세주소 합쳐서 표시) |
| 전화번호 | 항상 표시 |

> 자격상실신고 사업장 정보에는 단위사업장 명칭, 영업소 명칭 항목이 없음 (자격취득신고와 다름)

### 3.3 직원 목록

- 신고에 포함된 직원 목록을 순서대로 표시
- 각 항목: 성명 + 사번 표시
- 클릭 시 해당 직원이 선택 상태가 되어 우측 패널에 상세 내역 표시
- **초기 선택 상태**: 첫 번째 직원이 기본 선택됨

---

## 4. 우측 패널 — 직원 신고 상세

선택된 직원의 신고 내용을 섹션별로 표시. 모든 항목 읽기 전용.

### 4.1 기본 정보

| 항목 | 표시 조건 |
|------|----------|
| 성명 | 항상 표시 |
| 주민등록번호/외국인등록번호 | 항상 표시 |
| 전화번호 | 항상 표시 |
| 상실연월일 | 항상 표시 |

### 4.2 국민연금 (국민연금 신청한 경우에만 섹션 표시)

| 항목 | 표시 조건 |
|------|----------|
| 상실부호 | 항상 표시 (코드명으로 표시) |
| 초일취득·당월상실자 납부 | 신고서 작성 시 "초일취득·당월상실자 납부 희망"을 체크한 경우에만 표시, Badge "희망"으로 표시 |

### 4.3 건강보험 (건강보험 신청한 경우에만 섹션 표시)

| 항목 | 표시 조건 |
|------|----------|
| 상실부호 | 항상 표시 (코드명으로 표시) |
| 연간보수총액(해당연도) | 항상 표시 (천 단위 콤마 + "원", 없으면 "-") |
| 근무개월수(해당연도) | 항상 표시 ("N개월", 없으면 "-") |
| 연간보수총액(전년도) | 신고서 작성 시 "전년도 연말정산 미실시"를 체크한 경우에만 표시 |
| 근무개월수(전년도) | 신고서 작성 시 "전년도 연말정산 미실시"를 체크한 경우에만 표시 |

### 4.4 고용보험·산재보험 (고용보험 또는 산재보험 신청한 경우에만 섹션 표시)

| 항목 | 표시 조건 |
|------|----------|
| 상실사유코드 | 항상 표시 (코드명으로 표시) |
| 구체적 사유 | 항상 표시 (없으면 "-") |

**보수총액 표시 방식** — 고용보험과 산재보험을 모두 신청했고 신고서 작성 시 "두 보험 간 보수 다름"을 체크한 경우 분리 표시, 그 외에는 통합 표시:

#### Case A: 분리 표시 (고용보험·산재보험 모두 신청 + 두 보험 간 보수 다름 체크)

| 항목 | 표시 조건 |
|------|----------|
| 고용보험 보수총액(해당연도) | 항상 표시 (천 단위 콤마 + "원", 없으면 "-") |
| 산재보험 보수총액(해당연도) | 항상 표시 (천 단위 콤마 + "원", 없으면 "-") |
| 고용보험 보수총액(전년도) | 신고서 작성 시 "전년도 보수총액 신고 미실시"를 체크한 경우에만 표시 |
| 산재보험 보수총액(전년도) | 신고서 작성 시 "전년도 보수총액 신고 미실시"를 체크한 경우에만 표시 |

#### Case B: 통합 표시 (그 외)

| 항목 | 표시 조건 |
|------|----------|
| 보수총액(해당연도) | 항상 표시 (천 단위 콤마 + "원", 없으면 "-") |
| 보수총액(전년도) | 신고서 작성 시 "전년도 보수총액 신고 미실시"를 체크한 경우에만 표시 |

## 5. 충돌 지점 정리

| 항목 | 내용 | 비고 |
|------|------|------|
| 코드값 표시 방식 | 코드 번호가 아닌 코드명으로 표시 | 예: "11" → "퇴직" |
| 사업장 정보 항목 | 단위사업장 명칭, 영업소 명칭 없음 | 자격취득신고 상세와 다름 |
| 초일취득·당월상실자 납부 | "납부" 아닌 "희망" Badge로 표시 | |
| 전년도 항목 표시 조건 | 건강보험: 전년도 연말정산 미실시 체크 시 표시 / 고용·산재: 전년도 보수총액 신고 미실시 체크 시 표시 | 각 섹션별 체크 항목이 다름 |
