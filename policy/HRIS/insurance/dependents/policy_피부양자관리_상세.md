# 피부양자 관리 상세 화면(HRIS_INS_DEP_03) - 정책서

---

## 1. 화면 개요

- **화면 ID**: HRIS_INS_DEP_03
- **화면명**: 피부양자 관리 상세
- **목적**: 신고 완료된 피부양자 관리 신고 내역을 조회하는 읽기 전용 화면
- **주요 기능**: 신고 정보 조회, 사업장 정보 조회, 피부양자별 신고 상세 내역 조회
- **진입 경로**:
  - HRIS_INS_DEP_01 신고완료 상태 행 클릭 → 현재 탭에서 이동
  - HRIS_INS_DEP_01 '신고서 확인' 버튼 클릭 → 새 탭으로 열림
- **연관 화면**:
  - HRIS_INS_DEP_01 (피부양자 관리 목록) — 뒤로가기 클릭 시 이동

---

## 2. 레이아웃 구조

- **상단**: PageHeader (뒤로가기 + 화면명)
- **직원 정보 영역**: PageHeader 아래 가로 배치 영역 (성명 / 주민등록번호 / 전화번호) — 읽기 전용
- **중앙**: 좌측 패널 + 우측 패널 (좌우 분할 레이아웃)
  - **좌측 패널**: 신고 정보 + 사업장 정보 + 피부양자 목록
  - **우측 패널**: 선택된 피부양자의 신고 상세 내역 (읽기 전용)
- 수정 기능 없음 — 전체 화면 읽기 전용

---

## 3. 직원 정보 영역

PageHeader 아래, 좌우 패널 위에 가로로 배치된 읽기 전용 영역.

| 항목 | 표시 내용 |
|------|----------|
| 성명 | 신고 시 입력한 직원 성명 |
| 주민등록번호/외국인등록번호 | 신고 시 입력한 주민등록번호/외국인등록번호 |
| 전화번호 | 신고 시 입력한 전화번호(휴대폰번호) |

---

## 4. 좌측 패널

좌측 패널은 스크롤 가능하며, 신고 정보 → 사업장 정보 → 피부양자 목록 순서로 표시.

### 4.1 신고 정보

| 항목 | 표시 내용 |
|------|----------|
| 신고일자 | yyyy-MM-dd HH:mm:ss |
| 상태 | 작성중 / 신고완료 (일반 텍스트) |
| 팩스발송 | 작성중 상태: "-" / 신고완료 상태: "발송성공" (일반 텍스트) |
| 사업장 FAX | 신고 시 입력한 사업장 FAX 번호 |
| 공단 FAX | 신고 시 입력한 공단 FAX 번호 |

### 4.2 사업장 정보

| 항목 | 표시 조건 |
|------|----------|
| 관리번호 | 항상 표시 |
| 명칭 | 항상 표시 |
| 주소 | 항상 표시 (주소 + 상세주소 합쳐서 표시) |
| 전화번호 | 항상 표시 |

> 피부양자 관리 사업장 정보에는 단위사업장 명칭, 영업소 명칭 항목이 없음 (자격취득신고 상세와 다름)

### 4.3 피부양자 목록

- 신고에 포함된 피부양자 목록을 순서대로 표시
- 각 항목: 성명 + 구분(취득/상실) 표시
- 클릭 시 해당 피부양자가 선택 상태가 되어 우측 패널에 상세 내역 표시
- **초기 선택 상태**: 첫 번째 피부양자가 기본 선택됨

---

## 5. 우측 패널 — 피부양자 신고 상세

선택된 피부양자의 신고 내용을 섹션별로 표시. 모든 항목 읽기 전용.

### 5.1 기본 정보

| 항목 | 표시 조건 |
|------|----------|
| 구분 | 항상 표시 (취득 / 상실) |
| 취득일자 / 상실일자 | 항상 표시 (구분에 따라 라벨 변경) |
| 취득부호 / 상실부호 | 항상 표시 (구분에 따라 라벨 변경, 코드명으로 표시) |
| 피부양자 관계 | 구분이 "취득"인 경우에만 표시 (코드명으로 표시) |
| 성명 | 항상 표시 |
| 주민등록번호/외국인등록번호 | 항상 표시 |

### 5.2 장애인·국가유공자 (해당하는 경우에만 섹션 표시)

| 항목 | 표시 조건 |
|------|----------|
| 종별부호 | 항상 표시 (코드명으로 표시) |
| 등록일 | 항상 표시 |

### 5.3 외국인 (해당하는 경우에만 섹션 표시)

| 항목 | 표시 조건 |
|------|----------|
| 국적 | 항상 표시 (국가명으로 표시) |
| 체류자격 | 항상 표시 (코드명으로 표시) |
| 체류기간 | 항상 표시 |

---

## 6. 충돌 지점 정리

| 항목 | 내용 | 비고 |
|------|------|------|
| 코드값 표시 방식 | 코드 번호가 아닌 코드명으로 표시 | 예: "02" → "배우자" |
| 직원 정보 위치 | 좌측 패널이 아닌 상단 별도 영역에 표시 | 신고 화면과 동일한 레이아웃 유지 |
| 좌측 패널 구성 | 직원 목록 대신 피부양자 목록 | 피부양자 관리는 직원 1명 고정이므로 직원 목록 불필요 |
| 사업장 정보 항목 | 단위사업장 명칭, 영업소 명칭 없음 | 자격취득신고 상세와 다름 |
| 피부양자 관계 표시 조건 | 구분이 "취득"인 경우에만 표시 | 상실 신고에는 피부양자 관계 항목 없음 |
